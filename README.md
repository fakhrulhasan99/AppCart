# 📱 AppCart

## 🚀 Description
AppCart is a modern app store–like web application where users can explore, search, and install various popular apps. It provides a smooth user experience with real-time search, detailed app insights, and a personalized installation system using local storage. Users can also manage installed apps by uninstalling them anytime.

## ✨ Features
- 🔍 Live search (case-insensitive)
- 📊 App details with ratings chart (Recharts)
- 📥 Install apps with local storage support
- ❌ Uninstall apps from installation page
- ⚡ Dynamic routing with React Router
- 🔔 Toast notifications for actions
- ⏳ Loading spinner for better UX

## 🛠️ Technologies Used
- ⚛️ React
- 🌐 React Router
- 🎨 Tailwind CSS
- 📊 Recharts
- 🔔 React Toastify
- 💾 Local Storage API
